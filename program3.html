感染モデルのシミュレーション

身のまわりの事象をモデルとして捉えることで、複雑な現象をシンプルに理解したり予測したりすることができます。今回は、感染症の広がりを「SIRモデル」という数理モデルで表し、プログラミングによってシミュレーションを行います。

■ PythonでSIRモデルを書いてみよう

以下は、感染症の広がりをシミュレーションするSIRモデルのPythonコードです。

import numpy as np
import matplotlib.pyplot as plt

# パラメータ設定
N = 1000        # 総人口
I0, R0, V = 1, 0, 300
S0 = N - I0 - R0 - V
beta = 0.3      # 感染率
gamma = 0.1     # 回復率

# シミュレーション日数
days = 160
t = np.linspace(0, days, days)

# SIRモデルの初期リスト
S, I, R = [S0], [I0], [V]

# シミュレーション実行
for _ in range(1, days):
    next_S = S[-1] - beta * S[-1] * I[-1] / N
    next_I = I[-1] + beta * S[-1] * I[-1] / N - gamma * I[-1]
    next_R = R[-1] + gamma * I[-1]
    S.append(next_S)
    I.append(next_I)
    R.append(next_R)

# グラフ表示
plt.plot(t, S, label='Susceptible')
plt.plot(t, I, label='Infected')
plt.plot(t, R, label='Recovered')
plt.xlabel('Days')
plt.ylabel('Number of people')
plt.legend()
plt.title('SIR Epidemic Model')
plt.show()

■ Pythonコードの解説

  • N：総人口。シミュレーション対象の人数です。
  • I0：初期の感染者数。
  • V：最初からワクチン接種済みで免疫を持っている人数。
  • beta：感染率（1人が何人にうつすかの平均）
  • gamma：回復率（1日にどれくらいの割合が回復するか）
  • S, I, R：それぞれの日の感受性者・感染者・回復者の人数を記録
  • ループ内では、1日ごとの人数の変化を計算していきます。
