import numpy as np
import matplotlib.pyplot as plt

N = 1000
I0, R0, V = 1, 0, 300
S0 = N - I0 - R0 - V
beta, gamma = 0.3, 0.1

days = 160
t = np.linspace(0, days, days)

S, I, R = [S0], [I0], [V]

for _ in range(1, days):
    next_S = S[-1] - beta * S[-1] * I[-1] / N
    next_I = I[-1] + beta * S[-1] * I[-1] / N - gamma * I[-1]
    next_R = R[-1] + gamma * I[-1]
    S.append(next_S)
    I.append(next_I)
    R.append(next_R)

plt.plot(t, S, label='S')
plt.plot(t, I, label='I')
plt.plot(t, R, label='R')
plt.xlabel('Days')
plt.ylabel('Number of people')
plt.legend()
plt.title('SIR感染モデル')
plt.show()

このコードの流れ

  • 1〜4行目: 必要なライブラリを読み込みます。
  • 6〜9行目: 人数や感染率・回復率などの初期設定をします。
  • 11行目: 日数に対応するx軸（時間）のデータを作ります。
  • 13行目: S, I, R のそれぞれ初期値をリストとして定義します。
  • 15〜19行目: 1日ごとにS, I, Rがどう変化するかを計算し、毎回リストに追加します。
  • 21〜27行目: 最後に、変化の様子をグラフで表示します。
